# 7z-konmai

## Install
Put `konmai.dll` in  `C:\Program Files\7-Zip\Formats`


## Supported Formats (extract only)
- .bar

- .qar

- .mar (plain & encrypted)

- .arc

- .ifs (textures aren't converted to png)

- .2dx

- .s3p

- .cab/arcfile

- .d2
